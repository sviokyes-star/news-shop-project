# CS2 Hide Commands Plugin

Плагин для Counter-Strike 2, который скрывает консольные команды из чата.

## Возможности

- Скрывает команды, начинающиеся с `/`, `!`, `.` из публичного чата
- Скрывает команды из командного чата
- Легковесный и быстрый
- Не влияет на выполнение команд — только убирает из чата

## Установка

1. Соберите плагин:
   ```bash
   build.bat
   ```

2. Скопируйте файл `bin/Release/net8.0/HideCommandsPlugin.dll` в папку:
   ```
   game/csgo/addons/counterstrikesharp/plugins/HideCommandsPlugin/
   ```

3. Перезапустите сервер или выполните:
   ```
   css_plugins reload
   ```

## Как работает

Плагин перехватывает команды `say` и `say_team`, проверяет начало сообщения:
- Если начинается с `/`, `!` или `.` — сообщение не показывается в чате
- Команда всё равно выполняется, но невидима для других игроков

## Примеры

```
!ws          → скрыто в чате, команда работает
/top         → скрыто в чате, команда работает
.rank        → скрыто в чате, команда работает
привет всем  → показывается в чате (обычное сообщение)
```

## Требования

- CounterStrikeSharp
- .NET 8.0
