# CS2 Instant Respawn Plugin

Плагин для Counter-Strike 2, обеспечивающий мгновенное возрождение игроков после смерти.

## Возможности

- ✅ Автоматический респавн после смерти
- ✅ Настраиваемая задержка респавна (по умолчанию 0.5 сек)
- ✅ Возможность включать/выключать автореспавн
- ✅ Ручное возрождение конкретных игроков
- ✅ Полное восстановление здоровья при респавне
- ✅ Не респавнит ботов
- ✅ Не респавнит наблюдателей

## Команды

### Для администраторов (@css/root):

| Команда | Описание | Использование |
|---------|----------|---------------|
| `css_respawn` | Возродить игрока | `css_respawn [имя]` |
| `css_respawn_toggle` | Включить/выключить автореспавн | `css_respawn_toggle` |
| `css_respawn_delay` | Установить задержку респавна | `css_respawn_delay <секунды>` |

### Для всех игроков:

| Команда | Описание |
|---------|----------|
| `css_respawn_info` | Информация о настройках респавна |

## Установка

1. Соберите плагин:
   ```bash
   build.bat
   ```

2. Скопируйте файл `bin/Release/net8.0/InstantRespawnPlugin.dll` в папку:
   ```
   game/csgo/addons/counterstrikesharp/plugins/InstantRespawnPlugin/
   ```

3. Перезапустите сервер или выполните:
   ```
   css_plugins reload
   ```

## Примеры использования

### Включить/выключить автореспавн:
```
css_respawn_toggle
```
Переключает состояние автоматического респавна.

### Изменить задержку респавна:
```
css_respawn_delay 2
```
Устанавливает задержку 2 секунды перед респавном.

```
css_respawn_delay 0
```
Мгновенный респавн без задержки.

### Возродить конкретного игрока:
```
css_respawn Player123
```
Немедленно возрождает игрока с ником, содержащим "Player123".

```
css_respawn
```
Возрождает себя (если вызвано от имени игрока).

### Узнать текущие настройки:
```
css_respawn_info
```
Показывает статус автореспавна и текущую задержку.

## Как работает

1. **При смерти игрока:**
   - Плагин перехватывает событие смерти
   - Если автореспавн включен — ждёт заданную задержку
   - Возрождает игрока на случайном спавне его команды
   - Восстанавливает здоровье до 100 HP

2. **Условия респавна:**
   - Игрок должен быть в команде (CT или T)
   - Не работает для наблюдателей
   - Не работает для ботов
   - Игрок должен быть подключен к серверу

3. **Безопасность:**
   - Проверка валидности игрока перед респавном
   - Проверка состояния подключения
   - Защита от дублирования респавна

## Настройки по умолчанию

- **Автореспавн:** Включен
- **Задержка:** 0.5 секунды
- **Здоровье:** 100 HP

## Применение

Идеально подходит для:
- **Deathmatch серверов** — непрерывная игра
- **Aim карт** — быстрая тренировка
- **Fun серверов** — динамичный геймплей
- **Retake серверов** — быстрые раунды
- **1v1 арен** — мгновенные реванши

## Требования

- CounterStrikeSharp
- .NET 8.0
- Права администратора (@css/root) для управления настройками

## Совместимость

Плагин совместим с другими модификациями:
- Работает с кастомными спавнами (Spawn Manager)
- Совместим с оружейными плагинами
- Не конфликтует с таймерами карт
