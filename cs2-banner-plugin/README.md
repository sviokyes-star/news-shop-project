# Banner Manager

Плагин для размещения картиночных баннеров на стенах карт CS2. Баннер — это 3D-модель (Workshop-аддон) с изображением, которую плагин ставит в нужную точку карты.

## Требования

Для работы плагина у тебя уже должна быть подключена к серверу Workshop-модель с нужной картинкой (файл `.vmdl`). Плагин только размещает эту модель в мире — саму картинку-модель нужно найти или создать отдельно в Steam Workshop.

## Команды (только для админов с правами @css/root)

- `css_addbanner <путь_к_модели.vmdl> [расстояние]` — добавить баннер в точке, куда смотрит игрок (по умолчанию на расстоянии 100 юнитов). Пример:
  ```
  css_addbanner models/props/banners/my_banner.vmdl 150
  ```
- `css_removebanner <id>` — удалить баннер по его номеру на текущей карте
- `css_clearbanners` — удалить все баннеры на текущей карте
- `css_listbanners` — показать список всех баннеров текущей карты с координатами

## Как это работает

- Баннеры сохраняются отдельно для каждой карты в файле `banners.json`
- При смене карты (`OnMapStart`) плагин автоматически размещает все сохранённые для неё баннеры
- Баннер разворачивается лицом к игроку (в противоположную сторону от направления взгляда), чтобы сразу было видно, как он будет смотреться на стене
